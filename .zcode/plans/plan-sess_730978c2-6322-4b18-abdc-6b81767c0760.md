# Plan: Simplify to a Clean Astro Template (infrastructure preserved)

## Goal
Remove ONLY the decorative effects, the `/egg` easter egg, the email form, and the Google Forms wiring. Turn the homepage into a simple, clean hero. **Keep React, MDX, tailwind-merge, shadcn, Head.astro, CVA, motion, and Google Tag Manager** exactly as they are.

## Delete (only the things you named)
- `src/components/effects/glitch.css` + `GlitchScript.astro` — the glitch effect
- `src/components/effects/scramble.tsx` — the scramble-text effect
- `src/components/effects/EggTerminal.tsx` + `EggTerminal.css` — the egg terminal
- `src/pages/egg.astro` — the `/egg` route
- `src/components/forms/EmailForm.astro` — the email form **and** its Google Forms submission (the only place those IDs lived)
- `src/components/backgrounds/` (SmallCircles, LargeRed, shutters + their .css) — decorative animated backgrounds tied to the old aesthetic; nothing references them once the hero is simplified. *(If you'd rather keep the backgrounds, say so — easy to leave in.)*

The `effects/` and `forms/` directories end up empty and are removed.

## Rewrite (minimal)

Drop the `<GlitchScript>`, `<Scramble>`, `<EmailForm>`, and `<SmallCircles>` imports/markup. Keep the `astro-seo` block (already de-branded). Navbar stays hidden on home as before (or shown — your call; I'll keep current behavior).
2. **`src/config/globals.js`** — no change needed (GTM id already cleared to `""` with a TODO in the prior step; the integration code stays in Head/Layout).
